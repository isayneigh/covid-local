About

Purpose

There's a few sites out there already that provide extensive information on the COVID-19 virus. However, none are catered to the end user. COVID-19 Local attempts to provide a personalized experience so that users can get the information that matters most to them.


If you are interested in more extensive data you can check out the following sites:

  • Johns Hopkins University COVID-19 Dashboard
  • ncov2019.live
  • The Covid Tracking Project
